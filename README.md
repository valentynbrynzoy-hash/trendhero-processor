# 🚀 TrendHero Processor — Google Sheets Data Pipeline

Google Apps Script для автоматической обработки экспортов из [TrendHero](https://docs.google.com/spreadsheets/d/1gYmKUoZAedBYmePqfgfJgIBxWgtiuFtMkuDYvsgSWYo/edit?usp=sharing): фильтрация по ГЕО, извлечение контактов из bio, сортировка и форматирование результатов.

## 📋 Что делает

**Входные данные → одно нажатие → готовый к работе список с контактами**

1. **Фильтр по ГЕО** — оставляет только нужные страны из экспорта
2. **Парсинг контактов** — вытаскивает email и телефоны из поля `biography`
3. **Сортировка** — инфлюенсеры с контактами идут наверх
4. **Форматирование** — username становится кликабельной ссылкой на профиль, строки с контактами подсвечиваются зелёным

### Что извлекается из Bio

| Тип | Примеры |
|-----|---------|
| Email | `name@gmail.com`, `contact@brand.co.th` |
| Телефон с кодом страны | `+66 81 234 5678`, `+380-99-123-45-67` |
| Локальный номер | `0812345678`, `08 1234 5678` |
| Номер через дефис | `081-234-5678` |

Дубли в контактах в рамках одного профиля автоматически убираются.

## 🚀 Установка

### Быстрый старт (рекомендуется)

1. [Открой шаблон таблицы](https://docs.google.com/spreadsheets/d/1430GMpscRhiQzbiMvJAbBN23o_fpBMa_DETFnamb_dY/edit?usp=sharing)
2. **Файл → Создать копию** — сохранить себе на Drive
3. **Расширения → Apps Script** → удали дефолтный код → вставь `Code.gs` → сохрани
4. Закрой вкладку Apps Script, обнови таблицу
5. Появится меню **🚀 TrendHero** → запусти **setupSystem()** (один раз)

### 🔐 Авторизация скрипта

При первом запуске Google попросит дать разрешения.

1. Появится окно **«Требуется авторизация»** → нажми **Продолжить**
2. Выбери Google-аккаунт, к которому привязана таблица
3. Увидишь предупреждение **«Google не проверил это приложение»** — это нормально для личных скриптов
4. Нажми **«Дополнительные настройки»** → **«Перейти на страницу [название проекта] (небезопасно)»**
5. Нажми **Разрешить**

После этого разрешения сохраняются — повторно подтверждать не нужно.

### С нуля

1. Создай новую Google Таблицу
2. **Расширения → Apps Script** → вставь `Code.gs` → сохрани
3. Запусти функцию `setupSystem()` — она создаст все нужные листы
4. Обнови страницу — появится меню **🚀 TrendHero**

## ⚙️ Настройка

Все параметры — на листе **⚙️ Настройки**, менять прямо там:

| Параметр | По умолчанию | Описание |
|----------|-------------|---------|
| Разрешённые ГЕО | `TH` | ISO-коды через запятую: `TH, UA, KZ` |
| Колонка ГЕО | `country` | Имя колонки в данных TrendHero |
| Колонка Bio | `biography` | Имя колонки с описанием |
| Колонка ссылки | `ig_link` | Имя колонки со ссылкой на профиль |
| Извлекать контакты | ✅ | Чекбокс — вкл/выкл парсинг контактов |

## 📁 Структура таблицы

После `setupSystem()` в таблице появятся три листа:

```
⚙️ Настройки     ← меняй параметры здесь
📥 Входные данные ← вставляй экспорт из TrendHero сюда (с row 4)
📊 Результат      ← готовые данные после обработки
```

### Ожидаемые колонки во входных данных

```
username | full_name | ig_link | country | city | languages | biography | follower_count | following_count | media_count | general_er
```

Если в экспорте TrendHero другие названия — поправь соответствующие параметры в настройках.

## 📋 Меню

| Пункт | Действие |
|-------|---------|
| ▶ Запустить обработку | Фильтрация + парсинг + сортировка + запись в «📊 Результат» |
| ↕️ Contacts наверх | Пересортировать результат без повторной обработки |
| 🧹 Очистить результаты | Удалить данные из листа результатов |
| 📋 Очистить входные данные | Удалить данные из входного листа |

## 💡 Нюансы

- **ГЕО-коды** — двухбуквенные ISO 3166-1 alpha-2, заглавными: `TH`, `UA`, `US`, `ID`, `PH`. Список пустой = берутся все страны.
- **Названия колонок** — регистр не важен, `Country` и `country` — одно и то же.
- **Пересортировка** (`↕️ Contacts наверх`) — работает без повторной обработки, сохраняет форматирование и гиперссылки.
- **Очистка входных данных** — заголовки (строки 1–3) не трогаются, только данные.

## 🔧 Технические детали

- Runtime: Google Apps Script (V8 engine)
- Зависимости: нет, только встроенный SpreadsheetApp API
- ES5-совместимый JavaScript (требование Apps Script runtime)
- Тестировалось на экспортах TrendHero 2025–2026
